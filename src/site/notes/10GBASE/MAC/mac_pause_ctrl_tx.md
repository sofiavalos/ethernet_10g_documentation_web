## Description

implementa la transmision y gestión de marcos de control [MAC](Abbreviations.md#MAC), incluyendo tanto el flujo de control a nivel de enlace (LFC), como el flujo de control de prioridad (PFC).

| Generic name    | Type | Value | Description                                                 |
| --------------- | ---- | ----- | ----------------------------------------------------------- |
| MCF_PARAMS_SIZE |      | 18    | Tamaño de los parámetros del marco de control MAC en bytes. |
| PFC_ENABLE      |      | 1     | Habilita el control de flujo prioritario (PFC).             |
